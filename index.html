☁️ クラウド同期（GitHub Gist）

GitHubのプライベートGistにデータを保存して、PCとスマホで記録を共有できます。

最終同期: 未同期

📖 設定方法
  1. Fine-grained Personal Access Token を作成（Account permissions → "Gists" に Read and Write を付与）
  2. または classic token なら gist スコープを付与
  3. 取得したトークンを上の欄に貼り付け
  4. 初回端末で「Gist ID」を空欄のまま「⬆️ クラウドに保存」を押す → 新規Gistが作成され、Gist IDが自動入力されます
  5. もう片方の端末では同じトークン＋表示されたGist IDを入力して「⬇️ クラウドから読込」
  6. あとは自動同期をONにしておけば保存時に同期されます

⚠️ トークンはこの端末のブラウザに保存されます。共用端末では使用しないでください。

💾 ローカルバックアップ

クラウド同期とは別に、JSONファイルでバックアップできます。
