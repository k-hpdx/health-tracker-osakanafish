📖 セットアップ方法
  1. Firebase Console でプロジェクト作成
  2. Authentication → Sign-in method → Google を有効化
  3. Authentication → Settings → 承認済みドメインに k-hpdx.github.io を追加
  4. Firestore Database を作成（本番モード／東京リージョン推奨）
  5. Firestore Rules に以下を設定:
    match /users/{userId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == userId; }
  6. プロジェクト設定 → ウェブアプリを追加 → firebaseConfig をコピー
  7. 上の入力欄に貼り付けて「💾 設定を保存」
  8. 「Googleでログイン」でログイン → 2台目（スマホ等）も同じGoogleアカウントでログインすればデータ共有されます

⚠️ Firebase設定はこの端末のブラウザに保存されます。共用端末では使用しないでください。
